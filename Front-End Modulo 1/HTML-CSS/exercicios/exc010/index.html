Trabalhando com Listas

Listas Ordenadas

Uma lista ordenada é uma lista numerada, onde os itens são apresentados em uma sequência específica. Exemplo:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Trabalhar
  5. Estudar

Listas Não Ordenadas

Uma lista não ordenada é uma lista com marcadores, onde os itens não têm uma ordem específica. Exemplo:

  • Pão
  • Açucar
  • Chocolate
  • Arroz
  • Feijão
  • Macarrão
  • Carne
  • Frutas
