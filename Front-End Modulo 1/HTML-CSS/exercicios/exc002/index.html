Paragráfos e Quebras de Linha

Exemplo de paragrafos para estudos e estruturação de sites, para a criação desses paragrafos utilizamos a tag *p*

Caso precise de uma quebra de linha especifica utilizamos a tag br que significa breakrow, exemplo de uso: Lorem ipsum dolor sit amet consectetur adipisicing elit.
